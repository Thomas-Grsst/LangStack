Framework CSS utility-first pour un développement rapide

Qu'est-ce que Tailwind CSS ?

Tailwind CSS est un framework CSS utility-first qui permet de construire des designs personnalisés rapidement sans avoir à écrire du CSS personnalisé. Au lieu de classes prédéfinies comme Bootstrap, Tailwind fournit des classes utilitaires bas niveau.

Utility-First

Classes utilitaires pour tout

Personnalisable

Configuration totale

Performance

Purge CSS intégré

Installation

<!-- Via CDN -->
<script src="https://cdn.tailwindcss.com"></script>

<!-- Via npm -->
npm install -D tailwindcss

<!-- Fichier de configuration -->
npx tailwindcss init

<!-- tailwind.config.js -->
module.exports = {
  content: ["./src/**/*.{html,js}"],
  theme: {
    extend: {},
  },
  plugins: [],
}

<!-- Fichier CSS -->
@tailwind base;
@tailwind components;
@tailwind utilities;
