Cascading Style Sheets - Feuilles de style en cascade

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est un langage utilisé pour décrire la présentation des documents HTML et XML. Il permet de contrôler la mise en page, les couleurs, les polices et l'apparence générale des pages web.

Présentation

Contrôle l'apparence visuelle

Responsive

Adaptation aux différentes tailles d'écran

Animation

Création d'animations et transitions

Syntaxe de base

/* Commentaire CSS */
selecteur {
    propriete: valeur;
    autre-propriete: valeur;
}

/* Exemple */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 20px;
}

Sélecteurs CSS

/* Sélecteur d'élément */
p {
    color: blue;
}

/* Sélecteur de classe */
.ma-classe {
    font-size: 16px;
}

/* Sélecteur d'ID */
#mon-id {
    background: yellow;
}

/* Sélecteur multiple */
h1, h2, h3 {
    font-family: 'Arial', sans-serif;
}

/* Sélecteur d'enfant */
div > p {
    margin: 10px;
}

/* Sélecteur de descendant */
article p {
    line-height: 1.6;
}

/* Sélecteur d'attribut */
input[type="text"] {
    border: 1px solid #ccc;
}

/* Pseudo-classes */
a:hover {
    color: red;
}

button:active {
    transform: scale(0.98);
}

Flexbox

.conteneur-flex {
    display: flex;
    flex-direction: row; /* row | row-reverse | column | column-reverse */
    justify-content: center; /* flex-start | flex-end | center | space-between | space-around */
    align-items: center; /* stretch | flex-start | flex-end | center | baseline */
    flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */
    gap: 10px; /* Espace entre les éléments */
}

.element-flex {
    flex: 1; /* grow shrink basis */
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 200px;
    
    /* Alignement individuel */
    align-self: flex-start;
}

/* Exemple pratique */
.navigation {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem 2rem;
}

CSS Grid

.conteneur-grid {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* 3 colonnes */
    grid-template-rows: 100px auto 100px; /* 3 lignes */
    gap: 20px; /* Espace entre les cellules */
    grid-template-areas: 
        "header header header"
        "sidebar main aside"
        "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

/* Placement manuel */
.element-specifique {
    grid-column: 1 / 3; /* De la ligne 1 à 3 */
    grid-row: 2; /* Ligne 2 */
}

/* Grid implicite */
.grid-auto {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 1rem;
}

Couleurs et arrière-plans 

.couleurs {
    /* Formats de couleur */
    color: #ff0000; /* Hexadécimal */
    color: rgb(255, 0, 0); /* RGB */
    color: rgba(255, 0, 0, 0.5); /* RGB avec alpha */
    color: hsl(0, 100%, 50%); /* HSL */
    color: hsla(0, 100%, 50%, 0.5); /* HSL avec alpha */
    
    /* Arrière-plans */
    background-color: #ffffff;
    background-image: url('image.jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    
    /* Dégradés */
    background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
    background: radial-gradient(circle, #ff6b6b, #4ecdc4);
}

/* Variables CSS */
:root {
    --couleur-primaire: #3498db;
    --couleur-secondaire: #2ecc71;
    --espacement: 1rem;
}

.utilisation-variables {
    color: var(--couleur-primaire);
    margin: var(--espacement);
}

Animations et transitions 

/* Transitions */
.bouton {
    background-color: #3498db;
    transition: all 0.3s ease-in-out;
    transition-property: background-color, transform;
    transition-duration: 0.3s;
    transition-timing-function: ease;
}

.bouton:hover {
    background-color: #2980b9;
    transform: scale(1.05);
}

/* Animations */
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.element-anime {
    animation: fadeIn 0.5s ease-in-out;
    animation-delay: 0.2s;
    animation-fill-mode: both;
}

/* Animation continue */
@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

.loader {
    animation: spin 1s linear infinite;
}

Responsive Design

/* Media Queries */
/* Mobile First */
.conteneur {
    padding: 1rem;
    font-size: 14px;
}

/* Tablettes */
@media (min-width: 768px) {
    .conteneur {
        padding: 2rem;
        font-size: 16px;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .conteneur {
        padding: 3rem;
        font-size: 18px;
    }
}

/* Points de rupture courants */
@media (max-width: 640px) { /* Mobile */ }
@media (min-width: 641px) and (max-width: 768px) { /* Tablette */ }
@media (min-width: 769px) and (max-width: 1024px) { /* Petit desktop */ }
@media (min-width: 1025px) and (max-width: 1280px) { /* Desktop */ }
@media (min-width: 1281px) { /* Grand écran */ }

/* Orientation */
@media (orientation: landscape) {
    .conteneur {
        flex-direction: row;
    }
}

@media (orientation: portrait) {
    .conteneur {
        flex-direction: column;
    }
}
